Este proyecto es para crear una pagina web de venta de plantas.
Trabaje html, css y javascript.
Arme index.html de maneras sencilla: un header con una imagen como logo, el nombre del "negocio", el buscador y un navegador. El logo te redirige siempre a la pagian principal y con el navegador vas a diferentes paginas de la web.
Entre el header y el body, asi como entre el body y el footer puse un banner.
Dentro del body estan los productos, cada producto tiene un button para mas informacion o de compra. Al cliquear sobre mas informacion te direcciona a otra pagina con la informacion del producto, en este caso no encontre una api que me convenciera asi que es la misma informacion que se encuentra en la pagian principal. Al cliquear en comprar se añada al carrito y te aparece una confirmacion, al navegar hasta la pagina carrito se puede eliminar cualquier producto o incluso el total de productos del carrito.
El footer posee el logo de nuevo, el © y el navegador.
El CSS se me complico y seguro que hay alguna manera de hacerlo mas breve e incluso que se vea mejor al visitar la web desde diferentes formatos de dispositivo en este caso he de trabajar mejor con media query.
Fecha de hoy 14/12/2025, me propongo modificar css y carrito a futuro.
